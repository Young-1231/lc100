## 模板(739 每日温度)

```python
st = []                                    # 存下标,值递减
for i, x in enumerate(t):
    while st and t[st[-1]] < x:           # 我比栈顶大 → 它的右邻找到了
        ans[st.pop()] = i - st[-1]        # 注意 pop 之后再用 st[-1]
    st.append(i)
```

## 堆 Top-K

| 求 | 堆类型 | 大小 |
| --- | --- | --- |
| 第 K 大 / Top-K **大** | **小顶堆** | 维持 K(超出 pop) |
| 第 K 小 / Top-K **小** | **大顶堆**(取负) | 维持 K |
| 中位数(数据流) | 大顶 + 小顶 | 平衡两堆 |

## 堆比较器陷阱

`heapq` 是小顶堆;Python 的 ListNode/Tuple 比较时,若 key 相同会比较下一项。
解决:把唯一序号 idx 塞中间:`(val, idx, node)`。
