# 滑动窗口 模式表

| 类型 | 模板特征 | 例题 |
| --- | --- | --- |
| **定长滑窗** | 长度固定 K,移动一格丢一加一 | 438 异位词 |
| **变长滑窗-最长** | 满足条件就更新答案,违反则收缩 | 3 无重复子串、159 |
| **变长滑窗-最短** | 满足条件后用 while 收缩 | 76 最小覆盖、209 长度最小子数组 |
| **滑窗 + 单调队列** | 维护极值 | 239 滑窗最大值 |
| **前缀和 + 哈希** | 子数组求和(可负) | 560 子数组和=K |

## 通用模板(变长滑窗求最长)

```python
l = best = 0
state = {...}                    # 描述窗口状态
for r, x in enumerate(arr):
    state.add(x)
    while not_valid(state):
        state.remove(arr[l])
        l += 1
    best = max(best, r - l + 1)
```

## 三种"和为 K"题型对照

| 元素正负 | 求**子数组数量** | 求**最长长度** |
| --- | --- | --- |
| 全正 | 滑窗 | 滑窗 |
| 含负 | 前缀和+哈希(560) | 前缀和+哈希(每个 sum 只记**首次**位置) |

## 单调队列轨迹示例(LC 239,k=3)

```
arr = [1, 3,-1,-3, 5, 3, 6, 7]
i=0  dq=[0]                        (val: 1)
i=1  pop 0,dq=[1]                  (val: 3)
i=2  dq=[1,2]                      (val: 3,-1) → 输出 3
i=3  dq=[1,2,3]                    (val: 3,-1,-3) → 输出 3
i=4  popleft 1, pop 2/3, dq=[4]    (val: 5) → 输出 5
i=5  dq=[4,5]                      → 输出 5
i=6  pop 4,5, dq=[6]               (val: 6) → 输出 6
i=7  pop 6, dq=[7]                 (val: 7) → 输出 7
```
