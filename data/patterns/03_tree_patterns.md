## "返回单边贡献 / 全局取双边"通用模板

```python
def dfs(n):
    if not n:
        return 0
    l = max(0, dfs(n.left))   # 单边可能为负 → 砍掉
    r = max(0, dfs(n.right))
    self.best = max(self.best, n.val + l + r)   # 双边
    return n.val + max(l, r)                    # 返单边给父亲
```
