## 翻转链表三指针

```
prev=None      cur=head     nxt=cur.next
       ┌───┐         ┌───┐         ┌───┐
None ← │ 1 │ ─→ ⇒    │ 2 │ ─→  ... │ 3 │
       └───┘         └───┘         └───┘
       cur           nxt后续
```

## 快慢指针找中点

```
init: slow=head, fast=head
loop: while fast and fast.next:
          slow = slow.next
          fast = fast.next.next
偶数节点(找右中点):  1 -> 2 -> 3 -> 4   slow 停在 3
奇数节点:             1 -> 2 -> 3        slow 停在 2(中点)
```

## Floyd 入环点(LC 142)
设 head 到环入口距离 a,环长 L,入口到相遇点距离 b。
- fast 走 2(a+b),slow 走 a+b → fast - slow = a+b 是 L 的整倍数
- 因此 a = kL - b = (k-1)L + (L-b) = (k-1)L + c
- 让一个指针从 head、另一个从相遇点同步前进,会在入口相遇
